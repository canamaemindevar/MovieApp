# Pazarama iOS Bootcamp Movie App
Pazarama iOS Bootcamp final project using omdbapi. <br/> .
## Technologies and Libraries
- Protocol Oriented Programming
- Programmatic UI
- MVVM
- Delegate Pattern
- Dependency Injection
- Generic Network
- Compositional Layout
- UIPickerView
- Unit Tests
- UIKit
- SwiftUI
- UserDefults
